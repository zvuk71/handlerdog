                                    Muhammadatt • a year ago

                                    This may help someone else. I am using several Editable fields on the same page, submitting the result of each via Ajax. A couple things I wish I knew that may save the next person some time: (1) Be sure to use the Editable::begin rather than the Editable::widget if using multiple Editable fields on the same page. (2) I was able to use the following code to save and return the changes to any field on the page via ajax:

                                    if (isset($_POST['hasEditable'])) {
                                    // use Yii's response format to encode output as JSON
                                    \Yii::$app->response->format = \yii\web\Response::FORMAT_JSON;

                                    // save posted model attributes
                                    if ($model->load(Yii::$app->request->post()) && $model->save()) {

                                    //extract the class name
                                    $modelClassName = \yii\helpers\StringHelper::basename(get_class($model));
                                    //grab the post parameters array (as attribute=>value)
                                    $params = Yii::$app->request->post($modelClassName);
                                    //pull the first value from the array (there should only be one)
                                    $value = reset($params);

                                    // return JSON encoded output in the below format
                                    return ['output'=>$value, 'message'=>''];

                                    } else {
                                    // else if nothing to do always return an empty JSON encoded output
                                    // alternatively you can return a validation error
                                    return ['output'=>'', 'message'=>''];
                                    }
                                    }
